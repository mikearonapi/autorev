## 1. What Is AutoRev?

AutoRev is a **sports car intelligence platform** that helps enthusiasts through their entire journey—from finding the right car, to researching before buying, to owning smarter, to planning modifications.

### Core Philosophy

- **Unbiased**: No affiliate links, no dealerships paying for placement
- **For Everyone**: All budgets welcome ($25K-$200K+)
- **Expert-Driven**: Real ownership experience, not just spec sheets
- **AI-Enhanced**: Deep research via AL assistant
- **Transparent**: Clear tier benefits, no hidden features

### What It Does

| For... | AutoRev Helps By... |
|--------|---------------------|
| **Car Shoppers** | Matching them with cars based on what they care about most |
| **Buyers** | Providing safety data, pricing, best years to buy |
| **Owners** | Tracking maintenance, market value, VIN-specific info |
| **Tuners** | Planning builds with real dyno/lap data and parts compatibility |

## 6. AL — Your AI Assistant

AL (short for AutoLingo) is an **AI-powered car research assistant** built into the platform.

### What AL Can Do

| Capability | Example |
|------------|---------|
| **Find cars** | "What's a good track car under $50k?" |
| **Answer questions** | "What are the common issues with the E46 M3?" |
| **Research reliability** | "How reliable is the 997.2 Carrera S?" |
| **Compare cars** | "Cayman vs. Supra for a daily driver?" |
| **Recommend parts** | "What exhaust should I get for my BRZ?" |
| **Explain upgrades** | "Will coilovers make my car uncomfortable?" |

### How It Works

1. User types a question
2. AL uses tools to search the database, knowledge base, and parts catalog
3. AL responds with a helpful, conversational answer
4. Sources are cited when applicable

### Chat Limits

| Tier | Monthly Chats |
|------|---------------|
| Free | ~25 |
| Enthusiast | ~75 |
| Tuner | ~150 |

## 9. Common Questions

### Users & Tiers

**Q: How do users sign up?**
A: Click "Join" in the header → Choose a tier → Create account with email or Google

**Q: Are features really free during beta?**
A: Yes! All authenticated users get full access. Tier restrictions activate after beta.

**Q: How do I change someone's tier?**
A: Update their `subscription_tier` in `user_profiles` table in Supabase

### Data Management

**Q: How do I add a new car?**
A: Add to the `cars` table in Supabase with all required fields

**Q: How do I update car information?**
A: Edit directly in Supabase dashboard → Table Editor → cars

**Q: Where does the market value data come from?**
A: `car_market_pricing` table, populated from BaT, Cars.com, Hagerty

### AL Assistant

**Q: What if AL gives wrong information?**
A: AL uses your database as the source of truth. Update the data and AL will be correct.

**Q: Can AL access the internet?**
A: AL primarily uses your knowledge base. It can search for context but doesn't browse freely.

**Q: How do I add knowledge for AL?**
A: Add documents to `source_documents` and run the embedding script

### Technical

**Q: What if the site goes down?**
A: Vercel has 99.99% uptime. If Supabase is slow, the site uses local fallback data.

**Q: Is user data secure?**
A: Yes. HTTPS encryption, row-level security, Supabase enterprise security

**Q: Can I see analytics?**
A: Use Vercel Analytics (built-in) or add Google Analytics
